Política de Privacidade

Todo List App

Última atualização:

Resumo Executivo: O Todo List App armazena todos os dados exclusivamente no seu dispositivo. Não coletamos, compartilhamos ou transmitimos nenhuma informação pessoal. Seus dados são totalmente privados e seguros.

1. Informações que Coletamos

Nenhuma informação pessoal é coletada.

O Todo List App foi desenvolvido com foco total na privacidade do usuário. Não coletamos, armazenamos ou transmitimos nenhuma informação pessoal identificável.

1.1 Dados Armazenados Localmente

O aplicativo utiliza apenas o armazenamento local do dispositivo (localStorage) para salvar suas tarefas. Esses dados incluem:

  • Texto das tarefas que você cria
  • Status de conclusão das tarefas (concluída ou pendente)
  • Data de criação das tarefas

Importante: Todos esses dados permanecem exclusivamente no seu dispositivo e nunca são enviados para servidores externos.

2. Como Utilizamos as Informações

Como não coletamos informações pessoais, não há dados para utilizar ou processar externamente. Todas as funcionalidades do aplicativo operam exclusivamente com dados armazenados localmente no seu dispositivo.

3. Compartilhamento de Informações

Não compartilhamos informações com terceiros.

O Todo List App não compartilha, vende ou transmite dados para nenhuma empresa, organização ou serviço terceirizado. Seus dados permanecem privados e acessíveis apenas por você no seu dispositivo.

4. Armazenamento de Dados

4.1 Armazenamento Local

Todos os dados são armazenados localmente no seu dispositivo usando a tecnologia localStorage do navegador ou armazenamento nativo do sistema operacional (no caso de versões mobile).

4.2 Retenção de Dados

Os dados permanecem no seu dispositivo até que você:

  • Exclua manualmente as tarefas através do aplicativo
  • Limpe os dados do navegador/aplicativo nas configurações do dispositivo
  • Desinstale o aplicativo

5. Segurança dos Dados

Como todos os dados são armazenados localmente no seu dispositivo, a segurança depende das configurações de segurança do seu dispositivo e navegador. Recomendamos:

  • Manter seu dispositivo protegido com senha ou biometria
  • Manter o sistema operacional e navegador atualizados
  • Não compartilhar seu dispositivo com pessoas não autorizadas

6. Permissões do Aplicativo

O Todo List App solicita apenas as permissões mínimas necessárias para funcionar:

  • Internet: Apenas para carregar recursos iniciais (se aplicável). O aplicativo funciona completamente offline após o carregamento inicial.

O aplicativo não solicita acesso a:

  • Câmera
  • Microfone
  • Localização
  • Contatos
  • Arquivos do dispositivo (exceto armazenamento local)
  • Outras informações pessoais

7. Privacidade de Crianças

O Todo List App não coleta informações de crianças menores de 13 anos, pois não coleta informações de nenhum usuário. O aplicativo é seguro para uso por crianças, pois todos os dados permanecem no dispositivo e não são compartilhados.

Recomendamos que pais e responsáveis supervisionem o uso de aplicativos por crianças menores de 13 anos.

8. Alterações nesta Política de Privacidade

Podemos atualizar esta Política de Privacidade ocasionalmente. Recomendamos que você revise esta página periodicamente para estar informado sobre como protegemos seus dados.

Alterações significativas serão comunicadas através de uma atualização do aplicativo ou notificação na interface do aplicativo.

A data da última atualização está indicada no topo desta página.

9. Seus Direitos

Como todos os dados são armazenados localmente no seu dispositivo, você tem controle total sobre suas informações:

  • Acesso: Você pode visualizar todos os seus dados diretamente no aplicativo
  • Modificação: Você pode editar ou excluir qualquer tarefa a qualquer momento
  • Exclusão: Você pode excluir todas as tarefas ou desinstalar o aplicativo para remover todos os dados
  • Portabilidade: Como os dados estão no seu dispositivo, você tem acesso completo a eles

10. Conformidade com Regulamentações

Esta política de privacidade está em conformidade com:

  • Lei Geral de Proteção de Dados (LGPD) - Brasil
  • General Data Protection Regulation (GDPR) - União Europeia
  • Children's Online Privacy Protection Act (COPPA) - Estados Unidos

11. Contato

Se você tiver dúvidas ou preocupações sobre esta Política de Privacidade ou sobre como o aplicativo trata seus dados, entre em contato conosco através do repositório do projeto no GitHub.

Observação: Como não coletamos informações pessoais, não temos um banco de dados de usuários. Todas as questões relacionadas à privacidade podem ser direcionadas através do repositório público do projeto.
